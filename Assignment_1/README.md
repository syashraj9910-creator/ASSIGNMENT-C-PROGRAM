# Assignment_1: Student Marks Analyzer

## Objective
To develop a menu-driven C program for storing and analyzing the marks of N students using arrays and functions.

## Problem Statement
Write a menu-driven program that stores the marks of N students and performs the following operations:
- Display student marks
- Find highest marks
- Find lowest marks
- Calculate total and average marks
- Search a student's marks
- Sort marks in ascending order

## Algorithm
1. Start the program.
2. Input the number of students.
3. Read the marks of all students.
4. Display the menu.
5. Perform the selected operation.
6. Repeat until the user selects Exit.
7. Stop.

## Time Complexity
- Display: O(n)
- Highest: O(n)
- Lowest: O(n)
- Total & Average: O(n)
- Search: O(n)
- Bubble Sort: O(n²)

## Space Complexity
O(n)

## Sample Input
Enter number of students: 7

Marks:
78
65
90
55
82
70
88

## Sample Output
Highest Marks = 90

Lowest Marks = 55

Total Marks = 528

Average Marks = 75.43

Sorted Marks:
55 65 70 78 82 88 90
