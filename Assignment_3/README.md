# Assignment-03: Pointer Playground

## Objective
To understand the concepts of pointers in C programming, including memory addresses, dereferencing, pointer arithmetic, swapping values using pointers, and traversing arrays using pointers.

## Problem Statement
Develop a menu-driven C program that demonstrates pointer concepts. The program should:
- Display the memory address of a variable.
- Dereference a pointer to access the value.
- Swap two numbers using pointers.
- Traverse an array using pointers.
- Perform pointer arithmetic.

## Algorithm

1. Start the program.
2. Display the menu.
3. Read the user's choice.
4. Perform the selected operation:
   - Display memory address and value.
   - Dereference a pointer.
   - Swap two numbers using pointers.
   - Traverse an array using pointers.
   - Demonstrate pointer arithmetic.
5. Repeat until the user selects Exit.
6. Stop the program.

## Time Complexity

- Display Address: **O(1)**
- Dereference Pointer: **O(1)**
- Swap Two Numbers: **O(1)**
- Traverse Array: **O(n)**
- Pointer Arithmetic: **O(n)**

## Space Complexity

**O(n)** (for array traversal)

## Sample Input

```
Enter Choice: 3

Enter First Number: 10
Enter Second Number: 20
```

## Sample Output

```
After Swapping:

First Number = 20
Second Number = 10
```

## Learning Outcome

- Learned pointer declaration and initialization.
- Understood memory addresses and dereferencing.
- Implemented swapping using pointers.
- Traversed arrays using pointers.
- Learned pointer arithmetic.
- Developed a menu-driven C program using functions.
